Aula 21 - Tag Semântica

Problematização

  • A tag "div" define uma divisão ou seção de um documento Container de elementos HTML que podem ser estilizados com CSS ou manipulados com a linguagem JavaScript
  • Se utilizar "div" em páginas com muitas divisões, o código HTML pode se tornar de difícil compreensão para:
  • Navegador/browser
  • Motores de busca
  • Leitores de tela
  • Desenvolvedor
