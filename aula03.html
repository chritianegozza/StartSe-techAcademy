Como criar um documento HTML


Nessa aula aprendemos a como montar um arquivo html e baixar uma extensão, a estrutura básica de um arquivo html, como salvar e abrir no navegador, baixamos a extensão html Snippets e html css support


O que são tags HTML? As tags são usadas para informar ao navegador a estrutura do site. Ou seja: quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página.


Quais são as principais tags HTML?

comentário

Essa tag cria um comentário em qualquer parte do código-fonte. Útil para orientar pessoas em modificações futuras. Ou para relembrar sobre como cada linha funciona.


html

Essa tag está sempre no começo e no fim da página web. Ela limita onde o código começa e termina.


head

Utilizada para definir o começo e o fim do cabeçalho de um documento html.


body

Define o título do documento HTML. Esse é o nome que é exibido no topo da janela do navegador quando a página é aberta



tag html = tag principal


head = tag de cabeçalho


body = onde fica todos os elementos apresentados para os usuários
